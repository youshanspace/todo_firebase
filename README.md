#### 本地端測試

##### Node.js

- 使用版本: v16.16.0

##### 安裝 package

- 於終端機輸入下列指令，安裝相關 package

```
npm install
```

##### 安裝 package

- 於終端機輸入下列指令，運行前端頁面

```
npm start
```
